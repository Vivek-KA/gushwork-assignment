# Gushwork Frontend Assignment

This project is a responsive frontend website developed as part of the **Gushwork interview assignment**.  
The website demonstrates modern UI design, responsive layout techniques, and clean frontend development practices using **HTML, CSS, and JavaScript**.

---

##  Live Demo

Live Demo :  https://gushwork-assignment-lilac.vercel.app/  

---

 
## 🚀 Project Overview

The objective of this project was to build a **clean, responsive, and interactive landing page** based on the provided design reference.

The implementation focuses on:

- Responsive design for multiple screen sizes
- Clean UI layout
- Smooth user interactions
- Structured and maintainable code

---

## 🛠 Technologies Used

- **HTML5** – Semantic structure of the website
- **CSS3** – Layout, styling, animations, and responsiveness
- **JavaScript (Vanilla JS)** – Interactive functionality
- **Flexbox & Grid** – Modern layout techniques

---

## ✨ Features Implemented

- Responsive layout across different screen sizes
- Sticky navigation header
- Interactive card hover effects
- Carousel style application section
- Smooth scrolling interactions
- Clean and organized UI sections

Sections included in the page:

- Hero Section
- Trusted Companies Section
- Product/Features Section
- Applications Carousel
- Team Section
- Footer
